Weekly Reflection #1 (Module 01: Introduction to HTML)

Created By Selph

Finding a way to manage files and host the site pages re-introduced me to the struggles of a web developer. I learned that github.io could also host a site and provide me with the interfaces necessary to interact with files in the command line, in a desktop client GUI, and on the web interface. It was difficult to keep the versions in sync, leading to the need for a rebuild, which led to late work. I needed to find tools and methods to host and update the files, as well as become accustomed to using an HTML validator from W3.

Tools that supported my newly learned skills in this module include:

  • github.io from Linus Torvalds
  • Sublime Text Editor
  • The Duckett Textbook
  • The Professor!
